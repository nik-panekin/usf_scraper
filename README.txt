Written with Python 3.8.10

The script scrapes descriptions from the Ukrainian Startup Fund for each
project and then crawles the Web in search for contact data of the projects.

The results are saved to a CSV file.

How to use:
    python usf_scraper.py

Feel free to use and modify it as needed.

For dependencies refer to requirements.txt.

Данный скрипт выполняет парсинг описаний проектов из каталога Украинского Фонда
Стартапов (USF), а затем сканирует соответствующие сайты, пытаясь автоматически
найти контактные данные проектов.

Формат запуска:
    python usf_scraper.py
